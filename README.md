# Automotive Parts E-Commerce Website 🚗

A modern **automotive parts e-commerce web application** that allows users to browse vehicle components, add items to a shopping cart, and simulate a checkout process. The project demonstrates frontend UI design, backend API development, and cloud deployment.

---

## 🚀 Features

* Dynamic automotive product listing
* Shopping cart management
* Checkout flow with success page
* Backend API for fetching products
* Responsive and premium UI design
* Hero section with animated background / video
* About and Contact sections
* Mock payment integration

---

## 🛠 Tech Stack

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Data Storage

* JSON-based mock database

### Deployment

* Cloud deployment using Render

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/Safa-Akram/Automotive-e-commerce.git
```

Navigate to the project folder:

```
cd Automotive-e-commerce
```

Install dependencies:

```
npm install
```

Run the server:

```
node server.js
```

Open in browser:

```
http://localhost:3000
```

---

## 🌐 Cloud Deployment

The application can be deployed on cloud platforms like:

* Render
* AWS EC2
* Google Cloud Run

Example Render start command:

```

```
